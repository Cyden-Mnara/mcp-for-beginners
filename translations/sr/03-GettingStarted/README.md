- **2 Клијент**, у овој лекцији научићете како да напишете клијента који може да се повеже са вашим сервером, [до лекције](02-client/README.md)

- **3 Клијент са LLM**, још бољи начин писања клијента је додавањем LLM-а тако да може да "преговара" са вашим сервером шта треба да ради, [до лекције](03-llm-client/README.md)

- **5 stdio Transport Server** stdio транспорт је препоручени стандард за локалну MCP комуникацију између сервера и клијента, пружајући сигурну комуникацију засновану на потпроцесима уз уграђену изолацију процеса [до лекције](05-stdio-server/README.md)

- **6 HTTP стриминг са MCP (Streamable HTTP)**. [до лекције](06-http-streaming/README.md)

- **7 Коришћење AI Toolkit-а за VSCode** за конзумирање и тестирање ваших MCP клијената и сервера [до лекције](07-aitk/README.md)

- **8 Тестирање**. Овде ћемо се посебно фокусирати како можемо тестирати наш сервер и клијента на различите начине, [до лекције](08-testing/README.md)

- **9 Постављање у продукцију**. Овај поглавље ће погледати различите начине постављања ваших MCP решења, [до лекције](09-deployment/README.md)

- **10 Напредна употреба сервера**. Ово поглавље покрива напредну употребу сервера, [до лекције](./10-advanced/README.md)

- **11 Аутентификација**. Ово поглавље покрива како додати једноставну аутентификацију, од Basic Auth до коришћења JWT и RBAC. Препоручује се да почнете овде, а затим погледате Напредне теме у Поглављу 5 и извршите додатно ојачавање безбедности према препорукама у Поглављу 2, [до лекције](./11-simple-auth/README.md)

- **12 MCP домаћини**. Конфигуришите и користите популарне MCP хост клијенте као што су Claude Desktop, Cursor, Cline и Windsurf. Научите типове транспорта и решавање проблема, [до лекције](./12-mcp-hosts/README.md)

- **13 MCP инспектор**. Отклањајте грешке и тестирајте своје MCP сервере интерактивно користећи MCP инпектор алат. Научите како решавати проблеме са алатима, ресурсима и порукама протокола, [до лекције](./13-mcp-inspector/README.md)

- **14 Узорковање**. Креирајте MCP сервере који сарађују са MCP клијентима на задацима везаним за LLM (обавештење: застарело у кандидату за издање `2026-07-28`; још увек важеће за `2025-11-25`). [до лекције](./14-sampling/README.md)

- **15 MCP апликације**. Креирајте MCP сервере који такође одговарају уз упутства за кориснички интерфејс, [до лекције](./15-mcp-apps/README.md)

Протокол контекста модела (MCP) је отворени протокол који стандардизује начин на који апликације пружају контекст LLM-овима. Замислите MCP као USB-C порт за AI апликације – он пружа стандардизован начин повезивања AI модела са различитим изворима података и алатима.

## Циљеви учења

До краја ове лекције бићете у стању да:

- Подесите развојна окружења за MCP у C#, Java, Python, TypeScript и JavaScript
- Креирате и објавите основне MCP сервере са прилагођеним функцијама (ресурси, промптови и алати)
- Креирате хост апликације које се повезују на MCP сервере
- Тестирате и отклањате грешке у MCP имплементацијама
- Разумете уобичајене изазове у подешавању и њихова решења
- Повежете своје MCP имплементације са популарним LLM услугама

## Подешавање вашег MCP окружења

Пре него што почнете рад са MCP-ом, важно је припремити ваше развојно окружење и разумети основни ток рада. Овај одељак ће вас провести кроз почетне кораке подешавања како бисте обезбедили глатак старт са MCP-ом.

### Претпоставке

Пре него што зароните у развој са MCP-ом, уверите се да имате:

- **Развојно окружење**: За ваш изабрани језик (C#, Java, Python, TypeScript или JavaScript)
- **IDE/Уређивач**: Visual Studio, Visual Studio Code, IntelliJ, Eclipse, PyCharm или било који модеран уређивач кода
- **Менаџери пакета**: NuGet, Maven/Gradle, pip или npm/yarn
- **API кључеви**: За било које AI услуге које планирате користити у вашим хост апликацијама


### Службени SDK-ови

У наредним поглављима видећете решења направљена у Python, TypeScript, Java и .NET. Ево свих званично подржаних SDK-ова.

MCP пружа званичне SDK-ове за више језика (уклопљених са [MCP спецификацијом 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/)):
- [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk) - Одржава се у сарадњи са Microsoft-ом
- [Java SDK](https://github.com/modelcontextprotocol/java-sdk) - Одржава се у сарадњи са Spring AI-ом
- [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) - Званична TypeScript имплементација
- [Python SDK](https://github.com/modelcontextprotocol/python-sdk) - Званична Python имплементација (FastMCP)
- [Kotlin SDK](https://github.com/modelcontextprotocol/kotlin-sdk) - Званична Kotlin имплементација
- [Swift SDK](https://github.com/modelcontextprotocol/swift-sdk) - Одржава се у сарадњи са Loopwork AI-ом
- [Rust SDK](https://github.com/modelcontextprotocol/rust-sdk) - Званична Rust имплементација
- [Go SDK](https://github.com/modelcontextprotocol/go-sdk) - Званична Go имплементација

## Кључне поуке

- Подешавање MCP развојног окружења је једноставно уз SDK-ове специфичне за језик
- Изградња MCP сервера подразумева креирање и регистрацију алата са јасним шемама
- MCP клијенти се повезују са серверима и моделима како би искористили проширене могућности
- Тестирање и отклањање грешака су есенцијални за поуздане MCP имплементације
- Опције постављања крећу се од локалног развоја до решења у облаку
