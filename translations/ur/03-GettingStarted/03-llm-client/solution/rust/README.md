# اس نمونے کو چلانا

یہ LLM کلائنٹ نمونے کے لیے رسٹ حل ہے۔ آپ کو رسٹ ٹول چین انسٹال کرنے کی ضرورت ہے؛ دیکھیں [سرکاری انسٹال گائیڈ](https://www.rust-lang.org/tools/install).

کلائنٹ ایک ماڈل کو GitHub Models inference endpoint (`https://models.github.ai/inference/chat`) کے ذریعے کال کرتا ہے اور آپ کا GitHub ذاتی رسائی ٹوکن (PAT) `OPENAI_API_KEY` ماحولاتی متغیر سے پڑھتا ہے۔

> [!NOTE]
> اس ذخیرے میں دیگر حل `GITHUB_TOKEN` استعمال کرتے ہیں۔ رسٹ کے لیے، `OPENAI_API_KEY` کو اسی ویلیو پر سیٹ کریں تاکہ OpenAI کلائنٹ کی ترتیب سے میل کھائے۔

## -0- اپنا GitHub ٹوکن سیٹ کریں

```bash
# زی ایس ایچ/باش
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# پاورشیل
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- نمونہ تعمیر کریں

```bash
cargo build
```

## -2- نمونہ چلائیں

```bash
cargo run
```

کلائنٹ کیلکولیٹر MCP سرور شروع کرتا ہے، اس کی ٹول فہرست حاصل کرتا ہے، اور ماڈل (`openai/gpt-5-mini`) کو `add` ٹول کال کرنے کے لیے استعمال کرتا ہے۔ آپ کو ٹول کال ظاہر کرنے والا آؤٹ پٹ نظر آنا چاہیے (مثلاً، "Calling tool: add") اور اس کال کا نتیجہ۔
