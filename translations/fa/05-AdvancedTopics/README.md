# موضوعات پیشرفته در MCP

[![MCP پیشرفته: عامل‌های هوش مصنوعی امن، مقیاس‌پذیر و چندرسانه‌ای](../../../translated_images/fa/06.42259eaf91fccfc6.webp)](https://youtu.be/4yjmGvJzYdY)

_(برای مشاهده ویدیو این درس روی تصویر بالا کلیک کنید)_

این فصل شامل مجموعه‌ای از موضوعات پیشرفته در پیاده‌سازی پروتکل زمینه مدل (MCP) است، از جمله ادغام چندرسانه‌ای، مقیاس‌پذیری، بهترین شیوه‌های امنیتی و ادغام سازمانی. این موضوعات برای ساخت برنامه‌های MCP مقاوم و آماده تولید که بتوانند نیازهای سیستم‌های هوش مصنوعی مدرن را برآورده کنند، حیاتی هستند.

## مرور کلی

این درس مفاهیم پیشرفته در پیاده‌سازی پروتکل زمینه مدل را بررسی می‌کند، با تمرکز بر ادغام چندرسانه‌ای، مقیاس‌پذیری، بهترین شیوه‌های امنیتی و ادغام سازمانی. این موضوعات برای ساخت برنامه‌های MCP در سطح تولید که بتوانند نیازهای پیچیده در محیط‌های سازمانی را مدیریت کنند، ضروری هستند.

> **نگاهی به آینده:** چندین موضوع در زیر تحت تأثیر نسخه کاندید انتشار مشخصات MCP در تاریخ `2026-07-28` قرار دارند — زمینه‌های ریشه (5.4) و نمونه‌گیری (5.6) بر اساس ساختارهایی ساخته شده‌اند که نسخه کاندید آنها را منسوخ اعلام کرده است، و ویژگی تجربی وظایف که در ویژگی‌های پروتکل (5.16) ذکر شده است به یک افزونه اختصاصی وظایف منتقل می‌شود. برای جزئیات بیشتر به [چه چیزهایی در MCP تغییر می‌کند: نسخه کاندید 2026-07-28](../01-CoreConcepts/mcp-2026-07-28-release-candidate.md) مراجعه کنید.

## اهداف یادگیری

تا پایان این درس، شما قادر خواهید بود:

- قابلیت‌های چندرسانه‌ای را در چارچوب‌های MCP پیاده‌سازی کنید
- معماری‌های مقیاس‌پذیر MCP را برای سناریوهای با تقاضای بالا طراحی کنید
- بهترین شیوه‌های امنیتی مطابق با اصول امنیت MCP را بکار ببرید
- MCP را با سیستم‌ها و چارچوب‌های هوش مصنوعی سازمانی ادغام کنید
- عملکرد و اطمینان‌پذیری را در محیط‌های تولید بهینه‌سازی کنید

## نکات کلیدی

- پیاده‌سازی‌های چندرسانه‌ای MCP قابلیت‌های هوش مصنوعی را فراتر از پردازش متن گسترش می‌دهند
- مقیاس‌پذیری برای استقرارهای سازمانی حیاتی است و می‌توان آن را از طریق مقیاس‌پذیری افقی و عمودی رفع کرد
- تدابیر جامع امنیتی داده‌ها را محافظت می‌کنند و کنترل مناسب دسترسی را تضمین می‌کنند
- ادغام سازمانی با پلتفرم‌هایی مانند Azure OpenAI و Microsoft AI Foundry قابلیت‌های MCP را افزایش می‌دهد
- پیاده‌سازی‌های پیشرفته MCP از معماری‌های بهینه و مدیریت منابع دقیق بهره‌مند می‌شوند

## تمرین

یک پیاده‌سازی MCP در سطح سازمانی برای یک مورد استفاده خاص طراحی کنید:

1. نیازمندی‌های چندرسانه‌ای مورد استفاده خود را شناسایی کنید
2. کنترل‌های امنیتی لازم برای حفاظت از داده‌های حساس را مشخص کنید
3. معماری مقیاس‌پذیری طراحی کنید که بتواند بارهای متغیر را مدیریت کند
4. نقاط ادغام با سیستم‌های هوش مصنوعی سازمانی را برنامه‌ریزی کنید
5. گلوگاه‌های عملکرد احتمالی و راهکارهای کاهش آنها را مستند کنید
