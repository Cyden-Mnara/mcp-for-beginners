# اجرای این نمونه

این راه‌حل Rust برای نمونه کلاینت LLM است. شما نیاز به نصب زنجیره ابزار Rust دارید؛ راهنمای نصب رسمی را ببینید [official install guide](https://www.rust-lang.org/tools/install).

کلاینت از طریق نقطه پایان استنتاج مدل‌های GitHub (`https://models.github.ai/inference/chat`) با مدل تماس می‌گیرد و توکن دسترسی شخصی GitHub (PAT) شما را از متغیر محیطی `OPENAI_API_KEY` می‌خواند.

> [!NOTE]
> سایر راه‌حل‌ها در این مخزن از `GITHUB_TOKEN` استفاده می‌کنند. برای Rust، مقدار `OPENAI_API_KEY` را به همان مقدار تنظیم کنید تا سازگاری با پیکربندی کلاینت OpenAI حفظ شود.

## -0- توکن GitHub خود را تنظیم کنید

```bash
# زیرشِل/باش
export OPENAI_API_KEY="{{YOUR_GITHUB_PAT}}"
```

```powershell
# پاورشل
$env:OPENAI_API_KEY = "{{YOUR_GITHUB_PAT}}"
```

## -1- نمونه را بسازید

```bash
cargo build
```

## -2- نمونه را اجرا کنید

```bash
cargo run
```

کلاینت سرور MCP ماشین حساب را راه‌اندازی می‌کند، لیست ابزارهای آن را می‌گیرد و از مدل (`openai/gpt-5-mini`) برای فراخوانی ابزار `add` استفاده می‌کند. باید خروجی‌ای مشاهده کنید که فراخوانی ابزار را نشان می‌دهد (مثلاً "Calling tool: add") و نتیجه آن فراخوانی را.
